
🧠 Neurodegenerative Disease Early Risk Screening System

An AI-powered multi-disease risk screening system built using 
Machine Learning to assist in early detection of 
neurodegenerative diseases.

📌 Overview
This system predicts the risk of neurodegenerative diseases 
based on symptoms and clinical data using ML algorithms.

🛠️ Tech Stack
- Python
- Random Forest Algorithm
- Flask (Web App)
- HTML/CSS (Frontend)
- Streamlit
- Scikit-learn, Pandas, NumPy

✨ Features
- Multi-disease prediction
- Explainable AI (XAI) integration
- Confidence scoring
- Uncertainty-aware risk analysis
- Real-time ML prediction via web app
